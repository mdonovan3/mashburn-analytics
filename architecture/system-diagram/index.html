    Full System Diagram¶

    One page, two diagrams, both bigger and more granular than the summary pictures on Home and Architecture Overview — those show layers; these show the actual tables and model names inside each layer. Hover any node for a one-line description; click it to jump to the page that covers it in depth.

    Status legend is the same one used everywhere else on this site: ✅ Implemented · 🧩 Scaffolded · 📋 Planned.

    Whole system, source API to BI¶

    Every raw table for every source, both ingestion paths (mock + dlt), the orchestration pieces that will run dlt in production, all five raw_* datasets, staging/intermediate grouped by source, and all seven marts grouped by business domain.
